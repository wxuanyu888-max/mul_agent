## [1.0.0] - 2026-03-09

### Added

- Multi-agent collaboration system
- Tool kit architecture based on OpenClaw
- Skill system with SKILL.md format
- React + TypeScript frontend
- FastAPI backend
- Memory management system
- Command and hook system
- Python and TypeScript linting/formatting
- Pre-commit hooks with security scanning
- Mintlify documentation

### Changed

- Migrated tool system to SyncTool/AsyncTool pattern
- Restructured src/ directory to match OpenClaw layout
- Improved router and handler architecture

### Fixed

- Workflow page freeze issue
- Router handler registration
- Token usage tracking
