## 敏感信息处理

### 禁止事项

- 禁止在配置文件中硬编码密钥、密码、Token
- 禁止在日志中输出敏感信息
- 禁止在命令行参数中传递明文密码

### 敏感字段

| 字段类型 | 处理方式 |
|----------|----------|
| API密钥 | 环境变量读取 |
| 数据库密码 | 环境变量或密钥管理服务 |
| 用户密码 | 哈希存储，不存储明文 |

---

## 命令执行安全

### Bash 执行器

- 默认禁止危险命令（可配置）
- 命令超时强制中断
- 工作目录可限制

### 危险命令列表

| 命令 | 风险 | 建议 |
|------|------|------|
| rm -rf / | 数据丢失 | 禁止 |
| dd | 数据破坏 | 禁止 |
| curl \| sh | 代码注入 | 警告 |
| sudo | 权限提升 | 需要配置开启 |

---

## 配置文件安全

### 备份机制

- 修改配置前自动创建快照
- 快照保留最近N个版本

### 访问控制

- 配置文件权限：600（仅所有者读写）
- 敏感配置可加密存储

## 日志安全

### 日志级别

| 级别 | 内容 |
|------|------|
| DEBUG | 详细调试信息 |
| INFO | 正常流程信息 |
| WARNING | 警告信息 |
| ERROR | 错误信息 |

### 禁止记录

- 用户密码
- API密钥/Token
- 请求/响应的敏感字段
