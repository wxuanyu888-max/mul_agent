## 命令行参数

### --brain

启动核心大脑Agent，进入交互模式。

```bash
python main.py --brain
```

### --agent

使用指定Agent ID启动。

```bash
python main.py --agent <agent_id>
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| agent_id | string | 是 | Agent唯一标识 |

### --team

列出当前所有Agent成员及状态。

```bash
python main.py --team
```

### --route

手动触发指定的路由。

```bash
python main.py --route <route_name> [--params <json>]
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| route_name | string | 是 | 路由名称 |
| params | string | 否 | JSON格式参数 |

### --help

显示帮助信息。

```bash
python main.py --help
```
