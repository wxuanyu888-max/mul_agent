# 快速开始

本指南帮助你在 5 分钟内开始使用 Mul-Agent。

## 前置要求

- Python 3.13+
- Node.js 20+
- pnpm

## 安装

```bash
# 克隆仓库
git clone https://github.com/your-org/mul-agent.git
cd mul-agent

# 安装 Python 依赖
pip install -r requirements.txt

# 安装前端依赖
cd frontend && pnpm install
```

## 启动

```bash
# 启动 Agent 服务
python -m mul_agent.main
```
