# 评价改进

## 实施总结

### 完成的功能

1. **Grep 工具 - contextLines 参数**
   - 添加了 `contextLines` 参数，支持用户配置上下文行数
   - 默认值为 1，保持向后兼容

2. **Find 工具 - glob pattern 支持**
   - 添加了 `pattern` 参数，支持 glob 模式
   - 添加了 `ext` 参数，支持扩展名过滤
   - 修复了 `?` 通配符匹配逻辑

## 改进建议

1. **Grep semantic 模式**
   - 目前 semantic 模式的上下文由向量搜索决定，无法控制
   - 如需增强，可考虑在 Memory Manager 中添加 context 控制

2. **Find 工具 glob 解析**
   - 当前使用自定义正则实现，简单场景够用
   - 如需完整 glob 支持，可考虑使用 `micromatch` 或 `fast-glob` 库

3. **JSON 提取功能**
