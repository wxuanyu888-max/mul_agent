# 需求文档：PDF 和图片上传功能

## 需求背景

用户希望在聊天界面能够上传 PDF 文档和图片文件，让 AI Agent 能够处理这些文件内容。

## 详细描述

### 功能需求

1. **文件上传入口**
   - 在聊天输入框旁边添加上传按钮
   - 支持点击按钮选择文件
   - 支持拖拽文件到输入区域

2. **支持的文件类型**
   - 图片：PNG、JPG/JPEG、GIF、WebP
   - 文档：PDF

3. **文件预览**
   - 上传前显示缩略图预览
   - 支持删除已选择的文件
   - 支持一次上传多个文件

4. **消息发送**
   - 文件随消息一起发送给后端
   - 在聊天记录中显示文件图标/缩略图

### 期望结果

1. 用户可以点击上传按钮选择 PDF 或图片文件
2. 用户可以将文件拖拽到输入区域上传
3. 上传前显示文件预览（图片显示缩略图，PDF 显示图标）
4. 发送消息时文件一并发送
5. 聊天记录中显示已发送的文件

### 约束条件

1. 文件大小限制：单个文件最大 10MB
2. 需要后端支持文件上传和存储
3. 需要考虑向量化/OCR 处理（可选后续需求）

## 确认的方案

### 后端处理
- **原始文件存储**：文件保存在本地 storage/uploads/ 目录
- **内容提取**：
  - 图片：使用 OCR 提取文字内容
  - PDF：使用 PDF 解析提取文字
- **向量化**：使用 embeddings 模型生成向量（可选）

### 前端交互
- **按钮上传**：点击上传按钮打开文件选择器
- **拖拽上传**：拖拽文件到输入区域自动上传
- **预览**：图片显示缩略图，PDF 显示文件信息
- **多文件**：支持一次选择/拖拽多个文件
