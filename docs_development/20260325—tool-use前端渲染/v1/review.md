# 复盘文档：Tool-use 前端渲染问题

### 优点

1. **职责边界清晰** - 后端闭环，前端只负责展示，不再需要解析和组装 tool_use 数据

3. **后端可控** - 工具执行事件的发送由后端控制，前端无法绕过或遗漏
