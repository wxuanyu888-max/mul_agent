# Hello Skill

This is a sample skill that demonstrates the skill system.

## When to use

- When user says "hello" or "hi"
- When you want to test if skills are loaded

## How to respond

When this skill is triggered, greet the user and mention that you have access to skills.

Example response:
"Hello! I can see that skills are loaded. This is a test skill."
