## 记忆文件

- `storage/memory/notes/` - 笔记（自动索引）
- `storage/workspace/` - 工作空间文件（自动索引）

## 何时使用

**需要查找代码、实现细节或文件位置？**
→ 使用 `grep` 工具搜索工作空间文件

**需要回忆之前的决定、讨论或重要上下文？**
→ 使用 `memory` 工具的 search 功能

**正在处理任务并发现需要记住的重要内容？**
→ 使用 `memory` 工具的 write 功能

**在陌生领域开始新任务？**
→ 首先使用 `grep` 检查是否存在类似工作

## 关键区别

- **grep**：工作空间文件（代码库中**存在**的内容）
- **memory**：跨会话记忆（**已决定**或**已讨论**的内容）
