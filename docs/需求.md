# 需求

版本：v1.0
更新日期：2024-01-01

## 产品目标

构建一个能够自我进化的多Agent系统，核心大脑可以自主改写团队所有成员配置。该系统以CLI工具形式交付，具备动态团队分裂/协作能力，完全自主的进化权限。

聚焦用户价值：不需人工干预，Agent团队可自主完成复杂任务并持续优化自身能力。

## 目标用户

- 需要自动化任务执行能力的开发者
- 希望构建AI Agent团队的团队管理者
- 对自主进化AI系统感兴趣的研究者

## 功能范围

### 必做

- CLI命令行入口，支持多种启动模式
- 核心大脑Agent，具备自主决策能力
- 动态团队管理，支持Agent分裂创建新成员
- 记忆系统，支持短期/长期记忆和交接文档
- 工具层集成，支持Bash执行、Chrome MCP、Web Search MCP
- 配置文件管理，支持热重载和版本快照

### 可选（v1.0不做）

- Web UI界面
- 多核心大脑集群
- 分布式记忆存储

## 非功能需求

### 性能要求

- 核心大脑响应时间 < 2秒
- 命令执行器超时可配置，默认30秒

### 安全要求

- 配置修改前自动创建快照
- 危险命令可配置禁止列表
- 配置文件JSON Schema校验

### 兼容性

- Python 3.10+
- 支持Linux、macOS系统

## 核心规则

- 核心大脑可以自主修改所有配置文件（soul.json、user.json、skill.json、memory.json）
- Agent形态不固定，由核心大脑决定分裂/协作方式
- 记忆由Agent自行管理写入时机
- 交接文档由Agent自己生成，指定下一个Agent读取什么

## 命令行接口

| 命令 | 说明 |
|------|------|
| python main.py --brain | 启动核心大脑 |
| python main.py --agent <agent_id> | 指定Agent启动 |
| python main.py --team | 查看团队状态 |
| python main.py --route <route_name> | 手动触发路由 |

## 环境要求

- Python 3.10+
- 已配置MCP工具（如Chrome DevTools）
