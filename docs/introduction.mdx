# Mul-Agent 介绍

**Mul-Agent** 是一个自主的多 Agent 协作系统。

## 主要功能

- 🤖 **多 Agent 协作** - 支持多个 Agent 角色协同工作
- 🧠 **智能路由** - 自动路由任务到合适的 Agent
- 🛠️ **工具系统** - 内置丰富的工具集
- 📝 **记忆管理** - 持久化状态和记忆

## Agent 角色

| Agent | 职责 |
|-------|------|
| core_brain | 团队指挥官 |
| alice | 代码实现 |
| bob | 技术规划 |
| wangyue | 日常助手 |
