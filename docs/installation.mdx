# 安装指南

## 系统要求

- **Python**: 3.13 或更高版本
- **Node.js**: 20 或更高版本
- **pnpm**: 最新稳定版本

## 步骤 1: 安装 Python 依赖

```bash
pip install -r requirements.txt
```

## 步骤 2: 安装前端依赖

```bash
cd frontend
pnpm install
```

## 步骤 3: 配置环境变量

复制 `.env.example` 到 `.env`：

```bash
cp .env.example .env
```

编辑 `.env` 文件，添加你的 API 密钥。

## 步骤 4: 验证安装

```bash
# 运行测试
./run_tests.sh

# 运行代码质量检查
./scripts/quality-check.sh
```
